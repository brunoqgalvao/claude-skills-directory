Getting Started with Agent Skills

Learn how to install and use agent skills with Claude Code.

Agent Skills are reusable capabilities you can add to your AI coding assistant. This guide will help you get started.

Installation

Install skills using the CLI:

npx @skills/cli install codex

Usage

Once installed, skills are automatically available in your Claude Code sessions.
