Overview

This skill enables seamless integration with OpenAI's Codex CLI, providing powerful code analysis and refactoring capabilities directly within your Claude workflow.

Features

  • Automated Code Analysis: Analyze codebases for patterns, issues, and improvements
  • Smart Refactoring: AI-powered code refactoring with context awareness
  • Session Management: Maintain context across multiple interactions
  • Model Selection: Choose the optimal model for your task

Usage

Once installed, you can invoke Codex commands directly:

codex analyze ./src
codex refactor --file app.ts --pattern "extract-method"
